### 1. (연습문제 1.1-3) 합성함수의 정의역

**1단계 (What)**
* **계산:** $f(x)=\sqrt{x+1}$이고 $g(x)=\frac{1}{x}$일 때, $(f\circ g)(x) = f(g(x)) = \sqrt{\frac{1}{x}+1} = \sqrt{\frac{x+1}{x}}$ 입니다. 정의역은 근호 안이 0 이상이고 분모가 0이 아니어야 하므로 $\frac{x+1}{x} \ge 0$에서 $x \le -1$ 또는 $x > 0$ 입니다. $(g\circ f)(x) = g(f(x)) = \frac{1}{\sqrt{x+1}}$ 입니다. 정의역은 분모가 0이 아니고 근호 안이 0보다 커야 하므로 $x+1 > 0$에서 $x > -1$ 입니다.
* **근거 설명:** 무리함수는 루트 안이 0보다 크거나 같아야 하고, 유리함수는 분모가 0이 되면 안 된다는 가장 기본적인 약속을 적용했습니다.

**2단계 (So What)**
* **결과 해석:** 함수를 조립하는 순서가 바뀌면 결과물뿐만 아니라, 그 함수가 무탈하게 작동할 수 있는 '허용 범위(정의역)' 자체가 확 달라진다는 것을 직관적으로 보여줍니다. 믹서기에 얼음을 먼저 넣느냐, 과일을 먼저 넣느냐에 따라 갈리는 느낌이 다르듯, 수학에서도 결합 순서가 전체 모양과 범위를 완전히 바꿔놓습니다.

### 2. (연습문제 1.1-8) 합성함수 형태로 표현

**1단계 (What)**
* **계산:** 주어진 함수 $F(x)=\sqrt{\lvert x^3-1\rvert+3}$에 대하여, 안쪽 덩어리를 $g(x) = \lvert x^3-1\rvert+3$으로 두고, 이를 감싸는 바깥쪽 틀을 $f(x) = \sqrt{x}$로 둡니다. 두 함수를 겹쳐보면 $F(x) = f(g(x)) = (f\circ g)(x)$로 깔끔하게 분리됩니다.
* **근거 설명:** 복잡하게 엉킨 수식을 속함수(안쪽)와 겉함수(바깥쪽)로 쪼개어 단순화하는 합성함수의 성질을 이용했습니다.

**2단계 (So What)**
* **결과 해석:** 겉보기엔 굉장히 복잡해 보이는 수식이라도, 알맹이 부분과 껍데기 부분으로 살짝 나누어 보면 훨씬 이해하기 쉬워집니다. V자 모양으로 꺾이는 절댓값 그래프를 먼저 상상하고, 그 전체를 다시 둥글게 깎아주는 무리함수 틀에 쏙 집어넣는 식으로 그래프가 그려지는 과정을 단계별로 파악할 수 있게 해 줍니다.

### 3. (연습문제 1.2-8) 자연로그 역함수

**1단계 (What)**
* **계산:** 함수 $y = e^{x-1}$에서, 양변에 자연로그를 취하면 지수가 밑으로 내려와 $\ln y = x-1$이 됩니다. 이를 $x$에 대해 정리하면 $x = \ln y + 1$입니다. 마지막으로 $x$와 $y$의 자리를 맞바꾸면 역함수 $f^{-1}(x) = \ln x + 1$이 됩니다. 원래 함수의 치역이 양수였으므로, 새로운 함수의 정의역은 $x > 0$이 됩니다.
* **근거 설명:** 지수와 로그가 서로 묻고 답하는 역연산 관계라는 점을 이용하여 $x$와 $y$의 위치를 뒤집어 주었습니다.

**2단계 (So What)**
* **결과 해석:** 하늘 높은 줄 모르고 가파르게 치솟던 지수함수 그래프를 $y=x$라는 대각선 거울에 딱 비추어 대칭 이동시켰을 때, 완만하게 뻗어나가는 로그함수 그래프로 뒤바뀌는 기하학적 마법을 보여줍니다. 결과값을 보고 원래의 입력값을 거꾸로 추적해 낼 수 있게 됩니다.

### 4. (연습문제 1.3-1) 쌍곡함수

**1단계 (What)**
* **계산:** 공식에 따라 $\sinh x = \frac{e^x - e^{-x}}{2}$, $\cosh x = \frac{e^x + e^{-x}}{2}$ 입니다. 여기에 $\ln 2$와 $\ln 3$을 각각 집어넣으면 자연상수 $e$와 자연로그 $\ln$이 서로 지워집니다. $\sinh(\ln 2) = \frac{e^{\ln 2} - e^{-\ln 2}}{2} = \frac{3}{4}$ 이고, $\cosh(\ln 3) = \frac{e^{\ln 3} + e^{-\ln 3}}{2} = \frac{5}{3}$ 입니다. 두 값을 더하면 $\frac{3}{4} + \frac{5}{3} = \frac{29}{12}$ 입니다.
* **근거 설명:** 쌍곡함수가 지수함수로 이루어져 있다는 정의와, $e^{\ln x} = x$가 되어 깔끔한 숫자로 떨어지는 로그의 성질을 사용했습니다.

**2단계 (So What)**
* **결과 해석:** 이름은 거창해 보이고 전봇대 사이에 축 늘어진 전깃줄처럼 유려한 곡선을 그리지만, 결국 그 알맹이를 까보면 우리가 잘 아는 $e^x$라는 지수함수 조각들을 더하고 빼서 만든 것에 불과하다는 걸 수식으로 확인한 셈입니다. 복잡한 곡선을 다루기 쉬운 조각들로 분해해서 계산할 수 있게 해줍니다.

### 5. (연습문제 1.4-6) 역삼각함수

**1단계 (What)**
* **계산:** 괄호 안의 두 덩어리를 각각 각도 $\alpha$, $\beta$라고 해봅시다. 즉, $\alpha = \cos^{-1}\left(\frac{4}{5}\right)$, $\beta = \cos^{-1}\left(\frac{3}{5}\right)$입니다. 이는 직각삼각형에서 $\cos\alpha = \frac{4}{5}$, $\cos\beta = \frac{3}{5}$라는 뜻이므로, 피타고라스 정리에 의해 $\sin\alpha = \frac{3}{5}$, $\sin\beta = \frac{4}{5}$가 됩니다. 삼각함수 덧셈정리로 식을 풀면 $\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta = \left(\frac{3}{5}\right)\left(\frac{3}{5}\right) + \left(\frac{4}{5}\right)\left(\frac{4}{5}\right) = 1$ 입니다.
* **근거 설명:** 역삼각함수는 결국 '각도'를 의미한다는 개념을 직각삼각형 비율에 적용하고, 각도를 쪼개어 계산하는 사인 덧셈정리를 활용했습니다.

**2단계 (So What)**
* **결과 해석:** 빗변이 5, 밑변이 4인 직각삼각형의 각도와, 빗변이 5, 밑변이 3인 직각삼각형의 각도를 더해보니 정확히 $90^{\circ}$($\frac{\pi}{2}$)가 되어 사인값이 최고점인 1을 찍는 재미있는 기하학적 퍼즐을 보여줍니다.

### 6. (예제 1.4-15) 역쌍곡사인함수

**1단계 (What)**
* **계산:** $y = \sinh^{-1}x$라 두면 $x = \sinh y = \frac{e^y - e^{-y}}{2}$입니다. 양변에 $2e^y$를 곱하면 $(e^y)^2 - 2xe^y - 1 = 0$이 됩니다. 근의 공식으로 풀면 $e^y = x + \sqrt{x^2 + 1}$ (지수이므로 양수만 선택)이 됩니다. 양변에 자연로그를 취하면 $y = \ln(x + \sqrt{x^2 + 1})$로 증명됩니다.
* **근거 설명:** 쌍곡사인함수의 정의와 이차방정식의 근의 공식을 활용하여 변수를 뒤집어 정리했습니다.

**2단계 (So What)**
* **결과 해석:** 복잡해 보이는 역쌍곡선함수가 사실은 무리식과 로그함수의 조합으로 완벽하게 만들어질 수 있음을 보여줍니다. 원점을 부드럽게 가로지르는 원래 곡선을 대각선 거울($y=x$)에 비추면 이 자연로그 식의 둥근 그래프와 완벽하게 겹쳐집니다.

### 7. (연습문제 3.1-1) 연속이며 미분 가능한 함수

**1단계 (What)**
* **계산:** $x=0$에서의 연속성을 보려면 $\lim_{x\to0} x^2\sin\frac1x$를 살펴야 합니다. 사인은 항상 -1과 1 사이이므로 $-x^2 \le x^2\sin\frac1x \le x^2$이고, 조임 정리에 의해 극한은 0이 되어 연속입니다. 미분 가능성은 $\lim_{h\to0} \frac{h^2\sin\frac1h - 0}{h} = \lim_{h\to0} h\sin\frac1h = 0$이므로 미분계수가 0으로 존재합니다.
* **근거 설명:** 극한값을 구하기 위해 부등식을 샌드위치처럼 누르는 조임 정리(Squeeze Theorem)와 도함수의 기본 정의를 썼습니다.

**2단계 (So What)**
* **결과 해석:** $\sin\frac1x$는 원점 근처에서 미친 듯이 위아래로 진동합니다. 하지만 앞에 곱해진 $x^2$이 마치 푹신한 쿠션처럼 작용해서 그 요동치는 폭을 0으로 꽉 눌러버리기 때문에, 결국 그래프가 끊기지 않고 아주 매끄러운 평면(접선 기울기 0)을 가지며 부드럽게 통과하는 놀라운 모습을 보여줍니다.

### 8. (연습문제 3.1-11) 절댓값 함수의 도함수

**1단계 (What)**
* **계산:** 절댓값을 미분하기 쉬운 형태인 $|x| = \sqrt{x^2} = (x^2)^{1/2}$로 바꿉니다. 겉미분과 속미분을 하는 연쇄법칙에 따라 $\frac{1}{2}(x^2)^{-1/2} \cdot 2x = \frac{x}{\sqrt{x^2}}$가 됩니다. 분모가 다시 $|x|$로 돌아오므로, 정답은 $\frac{x}{|x|}$ (또는 $\frac{|x|}{x}$)가 됩니다.
* **근거 설명:** 뾰족한 절댓값 기호를 대수적인 제곱근 형태로 바꾼 뒤, 합성함수의 미분법을 적용했습니다.

**2단계 (So What)**
* **결과 해석:** V자 모양으로 꺾인 그래프의 오르막길(오른쪽) 기울기는 항상 1, 내리막길(왼쪽) 기울기는 항상 -1이라는 것을 하나의 깔끔한 수식으로 요약해 줍니다. 식에 $x=0$을 넣으면 분모가 0이 되어 터져버리는데, 이는 뾰족한 원점에서는 매끄러운 널빤지(접선)를 올려놓을 수 없다는 기하학적 사실을 그대로 반영합니다.

### 9. (연습문제 3.2-2) 음함수의 고계 도함수 I

**1단계 (What)**
* **계산:** $2x^2y-4y^3=4$를 미분하면 $4xy + 2x^2y' - 12y^2y' = 0$입니다. 점 $(2,1)$을 대입하면 $8 + 8y' - 12y' = 0$에서 $y' = 2$입니다. 식을 한 번 더 미분하여 $4y + 4xy' + 4xy' + 2x^2y'' - (24yy'^2 + 12y^2y'') = 0$을 얻고, 다시 값들을 모두 대입하면 $-4y'' = 60$이 되어 $y'' = -15$입니다.
* **근거 설명:** $x$와 $y$가 뒤섞인 식에서 $y$를 $x$의 덩어리로 보고 미분하는 음함수 미분법과, 곱으로 연결된 식을 풀기 위한 곱의 미분법을 썼습니다.

**2단계 (So What)**
* **결과 해석:** $y'=2$는 곡선이 그 점에서 위를 향해 가파르게 올라가고 있음을 뜻합니다. 하지만 두 번째로 미분한 값인 $y''=-15$가 큰 음수라는 것은, 곡선이 위로 올라가는 와중에도 고개를 푹 숙이듯 아래쪽으로 아주 강하게 휘어지고(위로 볼록) 있다는 역동적인 궤적을 상상하게 해 줍니다.

### 10. (연습문제 3.2-4) 음함수의 고계 도함수 II

**1단계 (What)**
* **계산:** $x^2+2y^3=xy+9$를 양변 미분하면 $2x + 6y^2y' = y + xy'$입니다. 점 $(3,0)$을 대입하면 $6 + 0 = 0 + 3y'$이므로 기울기는 $y' = 2$입니다. 위의 도함수 식을 한 번 더 미분하면 $2 + 12yy'^2 + 6y^2y'' = 2y' + xy''$이 됩니다. 알고 있는 값들을 싹 대입하면 $2 = 4 + 3y''$이 되어 $y'' = -\frac{2}{3}$를 얻습니다.
* **근거 설명:** 이전 문제와 똑같이 음함수 미분법과 곱의 미분법을 연달아 적용하여 기울기와 곡선의 휘어짐을 계산했습니다.

**2단계 (So What)**
* **결과 해석:** 점 $(3,0)$을 지나는 순간, 곡선은 기울기 2로 우상향하며 산을 오르는 중입니다. 동시에 2계 도함수가 음수이므로, 산등성이를 넘을 때처럼 팽팽한 접선보다 살짝 아래로 부드럽게 처지는 둥근 지붕 모양(위로 볼록)을 그리고 있음을 알 수 있습니다.

### 11. (연습문제 3.3-4) 역삼각함수와 로그의 도함수

**1단계 (What)**
* **계산:** 함수 $y=x\tan^{-1}x-\frac12\ln(1+x^2)$를 미분합니다. 앞은 곱의 미분을, 뒤는 합성함수 미분을 씁니다. $y' = \tan^{-1}x + x\frac{1}{1+x^2} - \frac{1}{2}\frac{2x}{1+x^2}$가 됩니다. 뒤의 두 분수 항이 형태가 똑같아 서로 지워지면서 아주 간단하게 $y' = \tan^{-1}x$만 남습니다.
* **근거 설명:** 역탄젠트 함수의 미분 공식과 자연로그의 연쇄법칙을 차례대로 적용하여 지저분한 항들을 깔끔하게 소거했습니다.

**2단계 (So What)**
* **결과 해석:** 이 길고 복잡한 식을 미분기에 넣고 돌렸더니 아주 단순명료한 $\tan^{-1}x$ 하나만 쏙 빠져나옵니다. 이는 나중에 적분을 배울 때 "도대체 어떤 모양을 깎아야 $\tan^{-1}x$가 나올까?"라는 질문에 대한 완벽한 정답 스케치를 미리 보여주는 셈입니다.

### 12. (연습문제 3.4-1) 평균값 정리의 적용

**1단계 (What)**
* **계산:** $f(x)=x^3-x^2-x+1$에서, 양 끝점의 높이는 $f(-1) = 0$, $f(2) = 3$입니다. 두 점을 직선으로 이은 평균 기울기는 $\frac{3-0}{2-(-1)} = 1$입니다. 이제 순간 기울기인 $f'(x) = 3x^2 - 2x - 1$이 1이 되는 지점 $c$를 찾습니다. $3c^2 - 2c - 2 = 0$을 근의 공식으로 풀면 $c = \frac{1 \pm \sqrt{7}}{3}$이 나오며, 두 값 모두 구간 $(-1, 2)$ 안에 쏙 들어갑니다.
* **근거 설명:** 굴곡 없이 매끄러운 다항함수이므로 평균값 정리를 적용할 수 있고, 이차방정식을 풀어 정확한 위치를 집어냈습니다.

**2단계 (So What)**
* **결과 해석:** 톨게이트 입구와 출구를 통과한 시간으로 잰 '평균 속도'가 시속 100km였다면, 중간에 과속 카메라가 없었어도 내 계기판 바늘이 정확히 시속 100km를 가리킨 순간이 무조건 있었다는 사실을 수학적으로 완벽하게 증명하는 재미있는 결과입니다.

### 13. (연습문제 3.4-6) 평균값 정리를 이용한 극한 판정

**1단계 (What)**
* **계산:** 구하려는 극한 $\lim_{x\to 0^+}\frac{3^x-3^{\sin x}}{x-\sin x}$를 보면, $f(t) = 3^t$라는 함수의 평균변화율 모양입니다. 평균값 정리에 따라 이는 어떤 점 $c$에서의 순간 기울기 $3^c \ln 3$과 같아집니다 ($\sin x < c < x$). $x$가 0으로 좁혀지면, 사이에 낀 $c$도 0으로 찌그러집니다. 따라서 $\lim_{c\to 0} 3^c \ln 3 = \ln 3$이 됩니다.
* **근거 설명:** 복잡한 분수를 평균값 정리를 이용해 도함수로 바꾼 뒤, 조임 정리를 통해 좁혀지는 구간 안의 극한값을 빼냈습니다.

**2단계 (So What)**
* **결과 해석:** 분자와 분모가 모두 0으로 꺼져가는 골치 아픈 극한 문제를, 두 점을 잇는 '기울기'라는 시각으로 바라본 발상의 전환입니다. 두 점 사이의 거리가 점 하나로 쪼그라들 때, 펑퍼짐했던 평균 기울기가 아슬아슬한 한 점의 접선 기울기로 마법처럼 스며드는 과정을 보여줍니다.

### 14. (예제 4.4-2) 함수의 오목성과 변곡점

**1단계 (What)**
* **계산:** $f(x)=x^4-x^2-2x+1$를 두 번 미분합니다. $f'(x) = 4x^3 - 2x - 2$, $f''(x) = 12x^2 - 2$입니다. $f''(x) = 0$이 되는 지점은 $x = \pm\frac{1}{\sqrt{6}}$입니다. 이 두 값 사이에서는 $f'' < 0$이므로 엎어진 밥그릇(위로 볼록) 모양이고, 그 바깥에서는 $f'' > 0$이므로 똑바로 놓인 밥그릇(아래로 볼록) 모양이 됩니다. 변곡점은 $x = \pm\frac{1}{\sqrt{6}}$일 때입니다.
* **근거 설명:** 두 번 미분한 값의 부호를 통해 곡선이 어느 쪽으로 굽는지(요철)를 판별하고, 부호가 반전되는 변곡점을 찾았습니다.

**2단계 (So What)**
* **결과 해석:** 한 번 미분한 게 속도라면, 두 번 미분한 것은 핸들을 꺾는 방향입니다. 여기서 구한 변곡점은 운전자가 S자 코스를 돌면서 좌로 꺾던 핸들을 우로 휙 돌리는 바로 그 짜릿한 찰나의 순간을 정확한 좌표로 잡아낸 것입니다.

### 15. (연습문제 4.4-8) l'Hopital 법칙을 이용한 극한

**1단계 (What)**
* **계산:** $\lim_{x \to 0}(1-\tan x)^{\frac{1}{x}}$에서, 식 전체를 $y$라 두고 양변에 자연로그를 씌웁니다. $\ln y = \frac{\ln(1-\tan x)}{x}$가 되어 분모 분자가 모두 0으로 가는 형태가 됩니다. 로피탈의 정리를 써서 위아래를 미분하면 $\frac{-\sec^2 x / (1-\tan x)}{1}$이 됩니다. $x=0$을 넣으면 값은 -1입니다. 우리가 구한 건 $\ln y$의 극한이므로, 원래 구하려던 $y$의 극한값은 $e^{-1} = \frac{1}{e}$입니다.
* **근거 설명:** 지수에 미지수가 있는 까다로운 극한을 자연로그를 통해 분수 꼴로 끌어내린 후, 로피탈의 정리를 써서 깔끔하게 해결했습니다.

**2단계 (So What)**
* **결과 해석:** 밑은 자꾸 1이 되려 하고 지수는 무한대로 커지려 하는 팽팽한 줄다리기입니다. 눈에 보이지 않는 이 힘겨루기에 로그라는 돋보기를 들이대 보니, 양쪽이 무승부를 기록하며 자연상수의 역수($1/e$)라는 아주 부드럽고 뜻밖의 지점에서 힘의 균형을 이룬다는 게 흥미롭습니다.

### 16. (기본공식) 부정적분 중요 공식

**1단계 (What)**
* **계산:** 적분의 기초 뼈대가 되는 핵심 공식 9가지입니다.
  1. $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ (예: $\int x^2 dx = \frac{x^3}{3} + C$)
  2. $\int \frac{1}{x} dx = \ln|x| + C$
  3. $\int e^x dx = e^x + C$
  4. $\int \sin x dx = -\cos x + C$ (예: $\int \sin(2x) dx = -\frac{1}{2}\cos(2x) + C$)
  5. $\int \cos x dx = \sin x + C$
  6. $\int \sec^2 x dx = \tan x + C$
  7. $\int \sec x \tan x dx = \sec x + C$
  8. $\int \frac{1}{1+x^2} dx = \tan^{-1} x + C$
  9. $\int \frac{1}{\sqrt{1-x^2}} dx = \sin^{-1} x + C$
* **근거 설명:** 미분 공식을 거꾸로 재생하여 원래의 함수(원시함수)를 복원해 내는 역연산 규칙들입니다.

**2단계 (So What)**
* **결과 해석:** 복잡하게 녹아내린 얼음 조각을 보고 원래 어떤 모양의 틀에서 얼렸는지 단번에 알아맞히게 해주는 조립 설명서와 같습니다. 이 기본 뼈대들을 잘 익혀두면 수식들이 이리저리 엉켜 있을 때도 길을 잃지 않고 원래 형태를 예쁘게 찾아갈 수 있습니다.

### 17. (FTC) 미적분학의 기본정리와 체인룰

**1단계 (What)**
* **계산:** '미적분학의 기본정리'는 넓이를 구하는 적분과 기울기를 구하는 미분이 서로 완전히 정반대의 과정임을 뜻합니다. 체인룰 적용: 적분 구간의 끝이 $g(x)$로 되어 있는 식 $\int_a^{g(x)} f(t) dt$를 $x$로 미분할 때는, 알맹이 함수 $f$에 위끝 $g(x)$를 집어넣고, 거기에 잊지 말고 위끝의 속미분인 $g'(x)$를 곱해주어 $f(g(x))g'(x)$를 만듭니다.
* **근거 설명:** 적분으로 정의된 덩어리를 미분할 때 겉껍질과 속알맹이를 차례로 미분하여 곱하는 연쇄법칙을 적용했습니다.

**2단계 (So What)**
* **결과 해석:** 울퉁불퉁한 그래프 아래의 2차원 '면적'을 구하기 위해 끙끙댈 필요 없이, 단순히 1차원 '높이'를 나타내는 원래 함수를 찾기만 하면 끝난다는 마법 같은 다리입니다. 넓이와 기울기라는 전혀 다를 것 같던 두 세계가 사실은 완벽하게 하나로 이어져 있음을 보여줍니다.

### 18. (연습문제 5.1-3a) 로그 적분법

### 21. (연습문제 5.2-3) 정적분의 가법성

### 23. (연습문제 5.3-5) 치환적분과 정적분

**2단계 (So What)**
* **결과 해석:** 그래프의 가로축을 스프링처럼 2배 빡빡하게 압축해서($2x$) 밀어버렸기 때문에($+2$), 롤러코스터의 레일 모양(함수 형태)은 똑같더라도 실제로 땅에 닿는 면적은 절반으로 쏙 줄어들었다는 것을 시각적으로 재미있게 보여줍니다.
